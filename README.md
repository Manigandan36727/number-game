# Number Guessing Game - Multiplayer

A real-time multiplayer number guessing game where two players try to guess each other's secret numbers.

## Features
- Real-time gameplay using Socket.io
- Create or join game rooms with 6-digit codes
- Players set secret numbers (0-100)
- Turn-based guessing with "Above" or "Below" hints
- Visual feedback for correct/wrong guesses
- Responsive design for mobile and desktop

## Prerequisites
- Node.js (v14 or higher)
- npm or yarn
